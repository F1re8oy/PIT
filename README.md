## Project Setup
`In terminal run commands:`

```sh
cd majaslapa/backend
docker-compose up -d
```

`Wait 5seconds and continue `

```sh
node server.js
```
`In second terminal run command:`

```sh
cd majaslapa
npm install
```

```sh
npm run dev
```
